<h1 align="center">ENTORNO VIRTUAL (BACKEND)</h1>

## 1. crea tu carpeta con:

```bash
  python3 -m venv .venv
```

## 2. activa el entorno:
   
```bash
   source .venv/bin/activate
```

## 3. instala flask dentro del entorno:

```bash
   pip install flask
```

## 4. verifica si se instalo:

```bash
  pip list
```

> ## Crea tu archivo
> app.py

> ## Ejecuta tu proyecto

```bash
  python3 app.py
```

> ## Desactiva el entorno
```bash
  deactivate
```
